# Teen Patti App

A Flutter application for a basic Teen Patti game interface.

## Overview
This app provides a responsive UI for a Teen Patti card game. It features a main menu screen and a game table screen simulating a multiplayer environment, scaling across mobile, tablet, and desktop viewports.

## Features
- **Responsive Layout:** Works smoothly on different device screen sizes.
- **Home Menu:** Easy navigation to join or create a table.
- **Game Table:** A visually structured round table displaying simulated player slots.

## Tech Stack
- **Framework:** Flutter
- **Language:** Dart

## Setup & Run
1. Make sure you have Flutter installed.
2. Clone the repository.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the app on your connected device or emulator.
